# TryOrder.cls
SeleniumBasicのWebDriverの制御が同期処理の兼ね合いで、条件によってエラー吐きまくる欠点を持っている為、その欠点を克服する為のクラス
WebDriverをラッピングしてSeleniumBasicの取り扱うパラメータやメソッド全てをクラス内で制御する。
具体的に言うと、同機ズレ時の再取得や読み込み待ちを常に行う。
また拡張機能としてWindowHandlerの制御やアラート制御、オブジェクト検索の関数なども用意
